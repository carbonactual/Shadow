# Shadow Scope

## Core modules

1. User-bound Shadow record
2. Learning records
3. Digital asset hash record
4. Account connection record
5. Revenue report
6. Campaign draft
7. Commerce draft
8. Risk gate
9. Chain adapter

## MVP flow

1. Create Shadow
2. Add learning instructions
3. Register asset hash
4. Connect account
5. Create campaign or commerce draft
6. Record revenue signal
7. View report
