# Integrations

Shadow uses connector modules for social, media, commerce, affiliate and work platforms.

Initial interface targets:

- Instagram
- TikTok
- YouTube
- X
- LinkedIn
- Twitch
- Spotify
- Snapchat
- WhatsApp Business

Each connector should implement:

- connect account
- read profile status
- prepare content draft
- collect analytics
- return revenue signals where supported

Live publishing must respect platform terms and owner approval settings.
